Утилита для macOS, позволяющая скачивать фильмы и сериалы по прямым ссылкам (.m3u8, .mp4) вместе с субтитрами.

## Как скачать и установить

1. **[СКАЧАТЬ ПОСЛЕДНЮЮ ВЕРСИЮ (ZIP-архив)](https://github.com/Alex410023/Stream_Video_Downloader/releases/latest)**
2. Распакуйте скачанный `.zip` архив.
3. Перетащите появившееся приложение **Stream Video Downloader.app** в папку **Программы** (Applications).

---

## Важное примечание при первом запуске (macOS)
Так как приложение скачано из интернета, а не из App Store, система macOS может написать, что "файл поврежден" или "разработчик не установлен". Это нормальная защита системы.

**Чтобы открыть программу:**
1. Нажмите на иконку программы **ПРАВОЙ кнопкой мыши** (или двумя пальцами по тачпаду).
2. Выберите **«Открыть»**.
3. В появившемся окне подтвердите открытие. 
*(Это нужно сделать только один раз, далее программа будет запускаться обычным двойным кликом).*

---

## Особенности
- Автоматически склеивает видео и субтитры без потери качества (ffmpeg).
- Поддерживает одновременную загрузку очередей из нескольких серий.
- **Встроенное автообновление:** программа сама подскажет, когда выйдет новая версия, и обновится в один клик.
